# Markdown to HTML Converter

## Known Limitations

This implementation follows all the rules specified in the assignment document. The only requirement is that different block elements (headings, paragraphs, lists) must be separated by at least one blank line to be parsed correctly.

## How to Run Tests

1. Download all files and place them in the same directory
2. cd into that directory
3. Run: `conda activate md2html-env`
4. Run: `pytest`
